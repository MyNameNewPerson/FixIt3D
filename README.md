# FixIt3D - Платформа для 3D ремонта и хобби

## Описание
 FixIt3D — это обновленная платформа, объединяющая базу 3D-моделей для ремонта техники и товары для хобби. Проект включает в себя современный дизайн, интеграцию с маркетплейсами и карту мастеров 3D-печати.

## Основные возможности
- **Два режима:** Переключение между "Запчастями" и "Хобби" меняет контекст всего сайта.
- **Умный поиск:** Фильтрация "шуточных" моделей и строгая привязка к брендам в режиме ремонта.
- **Расширенная карточка модели:**
  - 3D-просмотр (если доступен файл).
  - Калькулятор стоимости печати (материал + заполнение).
  - Ссылки на покупку готовых деталей (Amazon, AliExpress).
  - Предложение ближайшего мастера 3D-печати.
- **Карта мастеров:** Интерактивная карта с возможностью заказать печать через WhatsApp/Telegram.
- **SEO-оптимизация:** Встроенные мета-теги и JSON-LD разметка.

## Как запустить проект

1. **Важно:** Убедитесь, что вы находитесь в ветке `feature/complete-renovation-v2`.
2. Запустите файл `start.bat`. Он автоматически:
   - Остановит старые процессы на порту 3000.
   - Запустит сервер.
   - Откроет браузер на `http://localhost:3000`.

**Примечание:** Если вы запускаете несколько версий проекта одновременно, убедитесь, что они используют разные порты. Моя версия использует порт **3000**.

## Технологии
- **Frontend:** HTML5, CSS3 (Modern Grid/Flexbox), Vanilla JavaScript.
- **Backend:** Node.js (Express) + Vercel-style API.
- **Data:** Python-парсеры для Thingiverse и Printables.
- **Maps:** Leaflet.js.
